# Smart Note Assistant

Smart Note Assistant is a simple and user-friendly web application that helps users create, manage, search, and store notes efficiently.  
It is built using **HTML, CSS, and JavaScript** and uses **Browser Local Storage** for data persistence.

---

## Features

-  Create and save notes
-  Auto-generate note title from content
-  Real-time word count
-  Search notes by title or content
-  Delete notes easily
-  Persistent storage using LocalStorage (data remains after refresh)
-  Responsive and clean UI

---

## Technologies Used

- **HTML** – Structure of the application  
- **CSS** – Styling and layout  
- **JavaScript** – Logic and interactivity  
- **LocalStorage** – Client-side data storage
